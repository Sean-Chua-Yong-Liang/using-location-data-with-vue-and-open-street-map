# Using Location Data with Vue.js and Open Street Map

[![Patreon](https://img.shields.io/badge/patreon-donate-blue.svg)](https://www.patreon.com/maoberlehner)
[![Donate](https://img.shields.io/badge/Donate-PayPal-blue.svg)](https://paypal.me/maoberlehner)

This is an example project for the following article: [Using Location Data with Vue.js and Open Street Map](https://markus.oberlehner.net/blog/using-location-data-with-vue-and-open-street-map/)

## Build Setup

```bash
# Install dependencies.
npm install

# Serve with hot reload.
npm run serve

# Build for production with minification.
npm run build
```
